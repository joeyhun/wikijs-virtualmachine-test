If it is occuring randomly then we require detailed cucm traces as well as  RisDC perfmon logs to check what is triggering the spike in CPU  

If the alerts are happening at a specific time of the day then it could be due to some scheduled activity like DRS backup, MWI sync, CDR Load etc. Please check the output of the following commands to see if there are any core dumps or other resource issues

utils core active list

utils diagnose test

utils service list
