The Tomcat SSL Keystore is located at C:\Program Files\Apache Software Foundation\Tomcat 7.0\ssl\tomcat7.jks

Open (Administrator) command line and navigate to C:\Program Files\Java\jre7\bin\

NOTE: The Keystore password is "changeit" (default amongst most Java keystores)

IMPORTANT: Remember to backup the keystore before making any changes!

To view all keystore certificates

Run: keytool.exe -list -v -keystore "C:\Program Files\Apache Software Foundation\Tomcat 7.0\ssl\tomcat7.jks"

(Optionally add -Alias certalias to view only one certificate)

To Delete a certificate in the keystore

Run: keytool.exe -delete -alias aliasname -keystore "C:\Program Files\Apache Software Foundation\Tomcat 7.0\ssl\tomcat7.jks"

Generate CSR for existing certificate in keystore

Run: keytool.exe -certreq -alias tomcat -keystore "C:\Program Files\Apache Software Foundation\Tomcat 7.0\ssl\tomcat7.jks" -file C:\tomcat.csr

Install new certificate in keystore

Run: keytool.exe -import -trustcacerts -alias tomcat -file certificate.pem -keystore "C:\Program Files\Apache Software Foundation\Tomcat 7.0\ssl\tomcat7.jks"
