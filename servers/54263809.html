The process changed from time to time.
Most reliably, the quickest way to upgrade Firmware/BIOS is by using the Repository Manager:

https://www.dell.com/support/article/us/en/04/sln283183/dell-emc-repository-manager-drm-guide-and-download?lang=en#download

After installation:

+ Add a repository
+ Select the systems - the models of server(s)
+ Select "linux" as the OS
+ Select "Firmware" and "BIOS" (and Drivers, if needed)
 - be sure to "Select All"

There should be several files and items listed for the repository.


Download the repository to a desired location, and export it to a bootable ISO.

Create a bootable usb key with Rufus if necessary.  Otherwise, use the KVM remote utility to boot the ISO and boot into the updates utility.
