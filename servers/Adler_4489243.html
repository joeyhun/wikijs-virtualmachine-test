TLS Remarks

  • Installed *.intranet certificate to be able to enable SSL on default port 443
  • Configured to use TLS 1.2 only (Using IISCrypto)
  • Manually added values to the registry to force .Net 2 and 4 to use TLS 1.2 (adding SystemDefaultTlsVersions value)
    • [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\v4.0.30319]
      "SystemDefaultTlsVersions"=dword:00000001
      [HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\.NETFramework\v4.0.30319]
      "SystemDefaultTlsVersions"=dword:00000001
