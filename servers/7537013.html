Phone is just configuring IP

  1. a.     Unplug phone so that it goes completely dead and plug back in.
  2. b.    If still just configuring IP, check that the Admin Vlan is set to 13

            2. press 2 Network configuration
            3. arrow down to Amin. VLAN Id
            4. press **# (this unlocks the phone)
            5. press Edit
            6. enter 13

            8. press Save
