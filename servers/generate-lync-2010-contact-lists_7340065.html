Generate Lync 2010 Contact Lists

 

1.) Navigate to working directory (on victoria.sau.southern.edu)

Go to C:\Lync_Scripts\Populate Contact Lists\

Open CMD and navigate to: C:\Lync_Scripts\Populate Contact Lists\

Open Powershell and navigate to: C:\Lync_Scripts\Populate Contact Lists\

 

2.) Enable Active Directory Users to use Lync (If not already enabled)

Run powershell: .\AddgroupfromAD.PS1 [Active Directory Group name]

 

3.) Create Backup of Lync 2010 contact lists

Run: dbimpexp.exe /hrxmlfile:backup.xml

 

4.) Cleanup previous runs

Delete all files in C:\Lync_Scripts\Populate Contact Lists\contacts\

5.) Generate txt file of all contacts

Run powershell: .\GetContactlist.ps1 [Active Directory Group name] (You can list several groups)

(This will populate C:\Lync_Scripts\Populate Contact Lists\contacts\)

 

6.) Generate new Contact lists

Run: CSContactImporter.exe

Choose “Batch” configure settings like below and then click generate

 

 

Read C:\Lync_Scripts\Populate Contact Lists\contacts\BatchImport.log for any errors

7.) Import the merged xml into Lync 2010

run: dbimpexp.exe /import /hrxmlfile:backup_Merged.xml /restype:all
