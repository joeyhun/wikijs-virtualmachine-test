If a machine is running Windows, but isn't accessible with RDP, it can be restarted with the following command:

shutdown /r /m \\computername

If it responds with the following error, use PSKill (available from Microsoft).
"A system shutdown is in progress.(1115)"

Similar to pulling the power plug:
pskill \\computername winlogon

If the server has gone unresponsive after installing updates, try running:
pskill \\computername trustedinstaller
